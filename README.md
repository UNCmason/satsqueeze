# SatSqueeze ⚡

Bitcoin image compressor for inscriptions on OP_NET L1.  
Fewer bytes. Fewer sats.

## What it does
Compresses inscription images to your exact KB target.  
Same resolution. Less sats spent on Bitcoin L1.

## Live App
Live App: https://satsqueeze.vercel.app

## Built for
vibecode.finance OP_NET Vibecoding Challenge  
#opnetvibecode @opnetbtc

## How to use
1. Open the live URL
2. Drop your images
3. Set target KB
4. Click Squeeze All
5. Download and inscribe on Bitcoin L1
